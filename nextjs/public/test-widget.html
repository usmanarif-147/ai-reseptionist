Setup Instructions

  1. Log in to the dashboard at localhost:3000
  2. Go to Widget Settings in your dashboard
  3. Copy your Business ID from the embed code shown there
  4. Replace YOUR_BUSINESS_ID_HERE in the script tag at the bottom of this file
  5. Refresh this page — the chat button should appear in the bottom right corner
